TabView

DOES NOT work in all pages

For those rare pages, a new tab will open, since Chrome prevents extensions from messing with internal pages like New Tab, Bookmarks, and this help page.

Set a keyboard shortcut.

Go to chrome://extensions. Navigate to hamburger menu (3 lines) at the top left and set a keyboard shortcut for TabView.

Only seen tabs are captured.

TabView can't display thumbnails for tabs that have never been clicked on. If a tab is missing a thumbnail, navigate to it or press the button to scan all tabs in the window.

  • Close TabView modal

  • Close individual tab

  • Go to previous tab

  • Highlight next tab ( Shift for previous)

  • Go to highlighted tab
